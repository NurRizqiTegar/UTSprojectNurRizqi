Workout Tracker adalah aplikasi web progresif yang dirancang untuk membantu pengguna memantau dan mengelola aktivitas kebugaran mereka sehari-hari. Aplikasi ini menyediakan antarmuka yang intuitif dan mudah digunakan untuk mencatat berbagai jenis latihan, melacak durasi, serta memperkirakan kalori yang terbakar selama sesi olahraga.

Dengan fitur seperti filter berdasarkan jenis olahraga dan intensitas, pengguna dapat dengan mudah mengorganisir serta menganalisis rutinitas kebugaran mereka.

Aplikasi ini mendukung dua mode penggunaan:
- **Mode Tamu:** pengguna dapat langsung mencoba tanpa registrasi.
- **Mode Profil Terdaftar:** menyediakan perhitungan kalori yang lebih akurat berdasarkan usia, berat badan, tinggi badan, dan jenis kelamin.
